# ⚙️ Getting Started

Clone the repository

```bash
git clone https://github.com/Gnanashree5/gnanashree-portfolio.git
```

Navigate into the project

```bash
cd gnanashree-portfolio
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
